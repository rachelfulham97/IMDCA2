Projects

4th year Project

The aim of my project is to implement a zero-trust architecture on a corporate network. The use case is to prevent employees from being able to access the network laterally. I am using active directory domain services to enforce group policy and conditional access on the network. A zero-trust architecture is where there is no inherent trust in the network. If a user wants access to a resource they must be authenticated each time and their authorization checked. A policy decision point will be created and implemented to either allow or deny access.

3rd year projects

As part of my internship I worked as part of a team to create a website hosted on the intranet within the company to display information about who is parked in the EV charging spaces. The website allowed for users to signup/login, then with a click of a button their name would be displayed on the digital signage screens in the office. After 3 hours they would get a warning (their name would flash on the screen) to let them know they have been charging for 3 hours and other people are in the queue to charge. As a part of this project we tested it against SQL injection, found vulnerabilities and changed the code to prevent them.
